# Hello World Murmuration — Soul

This is the smallest possible murmuration. It exists to prove the harness wake loop works end-to-end with real identity files read from disk.

## Evolutionary purpose

To demonstrate that a real, disk-backed identity chain can be loaded, parsed, validated, and threaded through the executor to an agent subprocess — with no hardcoded fallbacks.

## Bright lines

- The hello-world agent never calls an LLM
- The hello-world agent never writes to GitHub
- The hello-world agent never commits to any repo
- The hello-world agent only prints its wake summary and exits cleanly

## Values

Proof over persuasion. Observation over assumption. The simplest thing that actually works.
